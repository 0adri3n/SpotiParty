<h2>🛠️ Installation Steps:</h2>

<p>1. Download source code</p>

<p>2. Edit config.yaml with your data</p>

<p>3. Install requirements if needed</p>

```
pip install flask requests pyyaml
```

<p>4. Start SpotiParty !</p>

```
py server.py
```
